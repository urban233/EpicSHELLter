EpicSHELLter documentation
==========================
EpicSHELLter is a small and basic backup-creation software written in Python.


:doc:`Getting started<getting_started/getting_started>`
-------------------------------------------------------
New to *EpicSHELLter*? Check out the getting started guides and use cases.
They contain an introduction to the library and when to use it.

:doc:`API reference<api_reference/api_reference>`
-------------------------------------------------
The reference guide provides a comprehensive overview of the EpicSHELLter API,
including a detailed explanation of the functionality of the methods
and the parameters that can be utilized.
It assumes that you have a fundamental understanding of the key concepts.
